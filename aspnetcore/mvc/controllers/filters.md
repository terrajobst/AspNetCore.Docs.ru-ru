### <a name="filter-types"></a><span data-ttu-id="ed9c0-122">Типы фильтров</span><span class="sxs-lookup"><span data-stu-id="ed9c0-122">Filter types</span></span>

* <span data-ttu-id="ed9c0-130">[Фильтры действий](#action-filters) могут выполнять код непосредственно до и после вызова отдельного метода действия.</span><span class="sxs-lookup"><span data-stu-id="ed9c0-130">[Action filters](#action-filters) can run code immediately before and after an individual action method is called.</span></span> <span data-ttu-id="ed9c0-131">С их помощью можно управлять аргументами, передаваемыми в действие, и возвращаемым из него результатом.</span><span class="sxs-lookup"><span data-stu-id="ed9c0-131">They can be used to manipulate the arguments passed into an action and the result returned from the action.</span></span>
